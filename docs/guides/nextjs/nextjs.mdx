# Tigris and Next.js

Tigris is the easiest way to work with data in your Next.js applications.

- Use it as a scalable, ACID transactional, real-time backend for your
  serverless applications.
- Build data-rich features without worrying about slow queries or missing
  indexes.
- Seamlessly implement search within your applications with its embedded
  search engine.
- Connect serverless functions with its event streams to build highly
  responsive applications that scale automatically.
