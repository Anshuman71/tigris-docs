Subscribes to published events

### Synopsis

Streams events in real-time.

```shell
tigris subscribe {db} {collection} {filter} [flags]
```

### Examples

```shell
tigris subscribe testdb
```

### Options

```
  -h, --help          help for subscribe
  -l, --limit int32   limit number of results returned
```
