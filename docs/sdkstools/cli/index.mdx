Manages Tigris project and collections, read and write data, perform
transactions and setup Tigris locally, all from the command line.

## Tigris Installation

Here is [instructions](installation.mdx) how to install the `tigris` utility.

### Usage

Usage:
tigris [command]

### Available commands

- [tigris alter](commands/tigris_alter.md) - Alters collection
- [tigris backup](commands/tigris_backup.md) - Dumps documents and schemas to JSON files
- [tigris branch](commands/tigris_branch.md) - Working with Tigris branches
- [tigris completion](commands/tigris_completion.md) - Generates completion script for shell
- [tigris config](commands/tigris_config.md) - Configuration commands
- [tigris create](commands/tigris_create.md) - Creates project, collection, namespace or app_key
- [tigris db](commands/tigris_db.md) - Database related commands
- [tigris delete](commands/tigris_delete.md) - Deletes document(s)
- [tigris delete-project](commands/tigris_delete-project.md) - Deletes project
- [tigris describe](commands/tigris_describe.md) - Describes database or collection
- [tigris dev](commands/tigris_dev.md) - Starts and stops local development Tigris server
- [tigris docs](commands/tigris_docs.md) - Generates CLI documentation in Markdown format
- [tigris drop](commands/tigris_drop.md) - Drops collection or application
- [tigris generate](commands/tigris_generate.md) - Generating helper assets such as sample schema
- [tigris import](commands/tigris_import.md) - Import documents into collection
- [tigris insert](commands/tigris_insert.md) - Inserts document(s)
- [tigris invitation](commands/tigris_invitation.md) - Invitation management commands
- [tigris list](commands/tigris_list.md) - Lists projects, collections or namespaces
- [tigris login](commands/tigris_login.md) - Authenticate on the Tigris instance
- [tigris logout](commands/tigris_logout.md) - Logout from Tigris instance
- [tigris ping](commands/tigris_ping.md) - Checks connection to Tigris
- [tigris quota](commands/tigris_quota.md) - Quota related commands
- [tigris read](commands/tigris_read.md) - Reads and outputs documents
- [tigris replace](commands/tigris_replace.md) - Inserts or replaces document(s)
- [tigris restore](commands/tigris_restore.md) - restores documents and schemas from JSON files
- [tigris scaffold](commands/tigris_scaffold.md) - Scaffold new application for project
- [tigris search](commands/tigris_search.md) - Search related commands
- [tigris server](commands/tigris_server.md) - Tigris server related commands
- [tigris transact](commands/tigris_transact.md) - Executes a set of operations in a transaction
- [tigris update](commands/tigris_update.md) - Updates document(s)
- [tigris version](commands/tigris_version.md) - Shows tigris cli version
