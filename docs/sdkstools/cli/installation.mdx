# Installation

<Tabs>
<TabItem value="npm" label="npm">

```shell
npm i -g @tigrisdata/tigris-cli
```

</TabItem>
<TabItem value="macOS" label="macOS">

```shell
brew install tigrisdata/tigris/tigris-cli
```

Alternatively install via curl:

```shell
curl -sSL https://tigris.dev/cli-macos | sudo tar -xz -C /usr/local/bin
```

</TabItem>
<TabItem value="macOS arm64" label="macOS (arm64)">

```shell
brew install tigrisdata/tigris/tigris-cli
```

Alternatively install via curl:

```shell
curl -sSL https://tigris.dev/cli-macos-arm64 | sudo tar -xz -C /usr/local/bin
```

</TabItem>
<TabItem value="Linux" label="Linux">

```shell
curl -sSL https://tigris.dev/cli-linux | sudo tar -xz -C /usr/local/bin
```

</TabItem>
<TabItem value="Linux arm64" label="Linux (arm64)">

```shell
curl -sSL https://tigris.dev/cli-linux-arm64 | sudo tar -xz -C /usr/local/bin
```

</TabItem>
</Tabs>

You can check that utility is installed and works by running:

```shell
tigris version
```
