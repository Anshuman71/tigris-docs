# Documents

**Documents** referred to a type of collection which operates on documents. Data records are stored as JSON documents.
Documents are composed of field-value pairs. The value of the field can be any of the JSON data types.

![Tigris Event Streaming](/img/jsondoc.jpg)

This collection offers simple and intuitive APIs for storing, querying, and searching JSON documents. Strong transaction APIs are also offered for these collections. The
transaction provides strictly serializable isolation by default. As Tigris indexes all your data automatically, which
means these collections are both searchable and queryable by any field.
