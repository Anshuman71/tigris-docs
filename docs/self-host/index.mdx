Tigris Server is an open source database that fully supports Cloud and Kubernetes. Deploying Tigris to frequently used platforms such as AWS Elastic Kubernetes Service (EKS) or GCP's Google Kubernetes Engine should be straightforward.

You can find more information about how to deploy Tigris to these platforms under the corresponding menu items.
