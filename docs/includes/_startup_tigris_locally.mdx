### Install CLI

<Tabs className={tabbedItemTheme}>
<TabItem value="macOS" label="macOS">

```shell
brew install tigrisdata/tigris/tigris-cli
```

[Alternative ways of installation](/cli/installation.mdx).

</TabItem>
<TabItem value="Linux" label="Linux">

```shell
curl -sSL https://tigris.dev/cli-linux | sudo tar -xz -C /usr/local/bin
```

[Alternative ways of installation](/cli/installation.mdx).

</TabItem>
</Tabs>

### Start local Tigris instance

```shell
tigris dev start
```

Once this command has completed, Tigris will be available on port `8081`.
